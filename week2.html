Original paste (unchanged) 
Transfer Local Files to Windows Server
• Create Windows EC2 Machine
• Generate the password
• Create File in Local Machine
• Open Remote Desktop Connection
• Click on Show Options
• Go to Local Resources
• Click on More
• Check the Drivers
• Click on ok
• Click on Connect
• Now Copy the File & Paste the File in the EC2 Machine
Transfer Local Files to Linux Server
• Create Linux EC2 Machine
• Generate the PPK File
• Download & Install WINSCP:
https://winscp.net/eng/download.php
• Enter the Host Name & User Name
• Attach PPK File
• Click on ok
• Click on Login
• Click on Yes
• Drag and Drop the file in Linux server
• Now we can connect to Putty & Check using Linux Commands
